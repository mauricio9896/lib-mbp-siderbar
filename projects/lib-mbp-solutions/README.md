# lib-mbp-solutions

Libreria Angular con **dos funcionalidades principales**:

- **Sidebar**: menu navegable con items anidados, temas y modo colapsado.
- **Auth**: login UI, sesion persistida, permisos y guards.

## Requisitos

- Angular ^21
- Angular Material ^21 (MatIcon/MatRipple)
- Angular Router (si usas rutas en el sidebar)

## Instalacion

```bash
npm i lib-mbp-solutions
```

## Build

```bash
ng build lib-mbp-solutions
```
